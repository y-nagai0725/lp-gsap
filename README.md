## 概要
* 架空のブランド紹介ランディングページです。
* GSAPを使用したスクロール連動アニメーションを取り入れたいという思いから作成してみました。

## 公開URL
[https://alphadesign.mikanbako.jp/](https://alphadesign.mikanbako.jp/)

## 目的
* GSAPを使用したスクロール連動アニメーションの練習

## 使用技術
* HTML
* CSS
* JavaScript
* GSAP
* lenis（慣性スクロール実装に使用）
